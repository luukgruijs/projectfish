# projectfish

To use the application, define a config that can look like this:

```
    'port': process.env.PORT || 8000,
    'environment': 'dev',
    'database': 'mongodb://localhost/fish',
    'secret': 'fishIsGood',
    'name': 'John Doe',
    'email': 'johndoe@gmail.com',
    'password': 'test',
```
